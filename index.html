Four-Year Automated Course Planner

The Great Plan

GitHub Repo: automatic-course-planner

Project Description

A mix of interactive visualization that automates 4 years of course scheduling, tailored for first-year Oliners. Users input their major and other constraints to generate a personalized course plan.

Interactivity

Users interact with the project by selecting their major and indicating preferences such as study abroad or graduating early. They can refresh for different course schedules and save their plan as a CSV file.

Ethics

The project aims to assist first-year Oliners in planning their course schedules. Data collected are non-sensitive user preferences. Potential malicious behavior, such as brute force attacks or data theft, is mitigated by not storing sensitive information.

Model

The project maintains user data (major, preferences) and generates a four-year course plan. This information is stored in classes like User and CoursePlan.

View

Input page allows users to select their major, choose courses, and set preferences. Output page displays the generated course plan and allows users to save it in various formats.

Controller

User input triggers methods in model classes to update the state and compute the course plan. The controller manages interactions between the user interface and the model.

Next Steps

Consider integrating a database of courses for each profile, allowing underclassmen to connect with upperclassmen. Explore additional resources for specific engineering majors.
